# Glossary

The full glossary of TASE.2, MMS, and tool terms is in
{doc}`../concepts/terminology`. Quick acronym reference:

| Term | Expansion |
|------|-----------|
| TASE.2 | Telecontrol Application Service Element 2 (IEC 60870-6) |
| ICCP | Inter-Control-Center Communications Protocol (the common name for TASE.2) |
| MMS | Manufacturing Message Specification (ISO 9506) |
| VMD | Virtual Manufacturing Device |
| ICC | Inter-Control-Center (domain) |
| SBO | Select-before-operate |
| CROB | Control Relay Output Block (DNP3 group 12) |
| DNP3 | Distributed Network Protocol 3 (IEEE 1815) |
| RTU | Remote Terminal Unit |
| PLC | Programmable Logic Controller |
| HMI | Human-Machine Interface |
| TLS | Transport Layer Security (Secure ICCP) |
| SSE | Server-Sent Events |
| CBB | Conformance Building Block |
| pcap | packet capture |
